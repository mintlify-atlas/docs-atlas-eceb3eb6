## Architecture

The Shopify App JavaScript SDK follows a layered architecture:

<Steps>
  <Step title="Core Layer (@shopify/shopify-api)">
    The foundation library that provides:
    - OAuth authentication flow
    - Admin API and Storefront API clients
    - Webhook handling and validation
    - Session management interfaces
    - Billing API integration
    - REST and GraphQL support
  </Step>
  
  <Step title="Framework Adapters">
    Framework-specific packages that wrap the core library:
    - **Remix**: `@shopify/shopify-app-remix`
    - **Express**: `@shopify/shopify-app-express`
    - **React Router**: `@shopify/shopify-app-react-router`
    
    Each adapter provides framework-specific utilities while maintaining consistent APIs.
  </Step>
  
  <Step title="Session Storage">
    Pluggable session storage adapters for different databases:
    - PostgreSQL
    - MySQL
    - MongoDB
    - Redis
    - DynamoDB
    - Prisma
    - In-memory (development only)
  </Step>
</Steps>

## Key Features

### Authentication & Authorization

<AccordionGroup>
  <Accordion title="OAuth 2.0 Flow">
    All packages support Shopify's OAuth 2.0 authentication flow with automatic token management:
    - Authorization code flow
    - Token exchange (Remix & React Router)
    - Online and offline access tokens
    - Automatic token refresh
  </Accordion>
  
  <Accordion title="Session Management">
    Built-in session handling with support for multiple storage backends:
    - Automatic session creation and validation
    - Scope verification
    - Token expiration handling
    - Online and offline session types
  </Accordion>
</AccordionGroup>

### API Clients

<CodeGroup>
```typescript GraphQL Client
import { shopifyApi } from "@shopify/shopify-api";

const shopify = shopifyApi({
  apiKey: process.env.SHOPIFY_API_KEY,
  apiSecretKey: process.env.SHOPIFY_API_SECRET,
  scopes: ["read_products", "write_products"],
  hostName: "myapp.example.com",
  apiVersion: "2024-01",
  isEmbeddedApp: true,
});

// Create GraphQL client
const client = new shopify.clients.Graphql({ session });
const response = await client.request(
  `query { shop { name } }`
);
```

```typescript REST Client
import { shopifyApi } from "@shopify/shopify-api";
import { restResources } from "@shopify/shopify-api/rest/admin/2024-01";

const shopify = shopifyApi({
  apiSecretKey: process.env.SHOPIFY_API_SECRET,
  hostName: "myapp.example.com",
  apiVersion: "2024-01",
  isEmbeddedApp: true,
  restResources,
});

// Use REST resources
const products = await shopify.rest.Product.all({
  session,
  limit: 10,
});
```
</CodeGroup>

### Webhook Handling

All packages provide webhook validation and processing:

```typescript
// Webhook validation
const isValid = await shopify.webhooks.validate({
  rawBody: requestBody,
  rawRequest: request,
});

// Process webhook
if (isValid) {
  await shopify.webhooks.process({
    rawBody: requestBody,
    rawRequest: request,
  });
}
```

## Choosing the Right Package

<Tabs>
  <Tab title="Starting Fresh">
    **Recommended: @shopify/shopify-app-remix**
    
    Remix provides the best developer experience with:
    - Modern React framework
    - Built-in routing and data loading
    - TypeScript support
    - Excellent performance
    
    Alternatively, **@shopify/shopify-app-react-router** if you prefer React Router v7.
  </Tab>
  
  <Tab title="Existing Express App">
    **Recommended: @shopify/shopify-app-express**
    
    Use Express adapter if you:
    - Have an existing Express.js application
    - Need maximum flexibility and control
    - Want to integrate with existing Express middleware
  </Tab>
  
  <Tab title="Custom Framework">
    **Recommended: @shopify/shopify-api**
    
    Use the core library if you:
    - Use a framework without a dedicated adapter
    - Need runtime-agnostic code (Cloudflare Workers, etc.)
    - Want to build a custom integration
  </Tab>
</Tabs>
